## Game Boy 実機一致 TODO リスト（CPU/PPU/APU 以外を中心）

- **Bus / I/O 周り**
  - 0xFF01–0xFF02: シリアル通信レジスタの実装（IF.SERIAL 割り込み含む）
  - 0xFF50–0xFF7F: その他 I/O レジスタの実装（必要なものを仕様に合わせて追加）
  - 0x38 アドレスに対する RE TI 強制パッチの削除またはオプション化（現在のハックを実機挙動から分離）

- **Timer / DIV / TIMA**
  - TIMA オーバーフロー時の「遅延リロード」挙動の再現
  - TAC 変更時の立ち上がり／立ち下がりエッジ依存のタイマ動作

- **PPU（タイミング以外）**
  - VRAM/OAM アクセス制限（モード 2/3 中の CPU 読み書き挙動）
  - DMA 転送を 160 サイクル相当で進める実装（現在は即時転送）
  - LCD OFF 中の LY/STAT/モード挙動を実機どおりにする

- **APU（追加で詰めるべき点）**
  - フレームシーケンサ（512Hz）による Length/Envelope/Sweep の正確な更新スケジュール
  - NR50 の音量 0 を「ミュート」として扱う実装（現在は 1–8 扱い）
  - Noise LFSR 7bit モードのビット合成手順を実機仕様に合わせて再検証
  - Wave RAM 読み取り制限ロジックと内部周期（*16）との整合性調整

- **セーブ周り**
  - MBC1 + バッテリ RAM の永続化（.sav ファイル読み書き）

- **テスト**
  - blargg の CPU/PPU/Timer テスト ROM を導入し、自動テストとして組み込む
  - 代表的な商用 ROM（ポケモン赤/緑, 星のカービィ等）での回帰テスト手順を整備する
